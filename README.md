# ICRIMST 2026 Conference Site

Standalone Vite + React export, ready to deploy on Vercel.

## Local development
```
npm install
npm run dev
```

## Build
```
npm install
npm run build
```
Output goes to `dist/`.

## Deploy to Vercel
Push this folder to a Git repo (or drag-and-drop deploy) and import it in Vercel.
Framework preset: **Vite**. Build command: `npm run build`. Output directory: `dist`.
The included `vercel.json` handles SPA client-side routing.
